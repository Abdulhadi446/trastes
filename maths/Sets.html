Operations Definitions

SymbolNameMeaning
∪UnionAll elements from both sets
∩IntersectionCommon elements only
-DifferenceElements in A but not in B
'ComplementNot in the set

Examples

  • If A = {1,2,3} and B = {3,4,5}
  • Then A ∪ B = {1,2,3,4,5}
  • And A ∩ B = {3}

Types of Sets

  • Empty Set (∅): Has no elements
  • Subset (⊆): All elements in B are also in A
  • Universal Set (U): Contains all sets
  • Finite: Can count elements
  • Infinite: Cannot count (like natural numbers)
