What is Square Root?

Square root of a number is a value that, when multiplied by itself, gives the original number.

If √x = y, then y × y = x

Symbol: √ is the square root symbol. Try the calculator below!

Properties of Square Roots

  • √(a × b) = √a × √b
  • √(a²) = a
  • √(a/b) = √a / √b
  • √0 = 0

Exam Tips

  1. Memorize squares from 1 to 30
  2. Check answer by squaring it
  3. For non-perfect squares, approximate
